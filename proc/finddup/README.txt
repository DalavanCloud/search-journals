Antes de executar o find_dup.py 

- Certificar que os os arquivos de configuração schema.xml e solrconfig.xml do índice scielo-articles estão atualizados com a última versão do repositório git

- Reiniciar o servidor tomcat do índice scielo-articles para usar as últimas configurações do índice

- Executar uma re-indexação completa do índice
